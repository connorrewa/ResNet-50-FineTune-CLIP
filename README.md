
# Steps for reproducing results
- Download Python 3.11.9
- Download COCO2014 dataset: https://www.kaggle.com/datasets/jeffaudi/coco-2014-dataset-for-yolov3

## Virtual Environment  on Windows OS:
py -3.11 -m venv .venv
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process -Force
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
pip install -r requirements.txt

## Directory Structure (expected)
```
root/
	coco_data/
		train2014/                # COCO train images
		val2014/                  # COCO val images
		annotations/              # Labels
	processed_data/
		train_data.pt             # Generated by preprocess_setup.py
		val_data.pt               # Generated by preprocess_setup.py
	saved_models/
		clip_resnet_v3.pth 
```

python preprocess_setup.py
